
# Factsolotl 

#### Capstone in progress, 126019

#### By _**Rachael Bouissey**_

## Description

- The Factsolotl MVP is a search by zip code feature that will show a user lead concentrations from demo data.

## Setup/Installation Requirements

* Clone the master branch from Github https://github.com/rbouissey/Factsolotl .
* From the project root folder run two commands in the terminal:
  - npm i
  - npm start

* View the project at the link provided in the terminal.
